## Как запустить

### Требования
- .NET 8 SDK (Windows)
- Visual Studio 2022 или VS Code с расширением C# Dev Kit

### Запуск через Visual Studio
1. Открыть папку `EventOrganizer`
2. Дважды кликнуть `EventOrganizer.csproj`
3. Нажать `F5` или кнопку "Запустить"

### Запуск через командную строку
```bash
cd EventOrganizer
dotnet run
```

## Функциональность приложения

### 1. Список мероприятий (DataGrid)
- Отображает: №, Название, Дата, Секция, Количество участников
- Автоматическая нумерация строк
- Выделение строки → показывает участников этого мероприятия

### 2. Добавление мероприятия
- Поля: Название, Дата (DatePicker), Секция (ComboBox)
- Доступные секции: Пленарная, Техническая, Научная, Мастер-класс, Воркшоп, Другое
- Валидация обязательных полей

### 3. Форма регистрации участника (правая панель)
- Поля: Имя, Фамилия, E-mail, Роль, Мероприятие
- Роли: Докладчик, Слушатель, Организатор, Модератор, Гость, Спонсор
- Проверка дублирования по e-mail в рамках мероприятия

### 4. ListBox участников
- Верхний ListBox (внизу слева): участники выбранного мероприятия
- Нижний ListBox (справа): все зарегистрированные участники
- Отображение: инициалы, ФИО, роль, мероприятие

### 5. Удаление
- Удаление мероприятия → автоматически удаляются все его участники
- Удаление отдельного участника из общего списка

## Разбор ключевых элементов кода

### ObservableCollection
```csharp
private ObservableCollection<Event> _events = new ObservableCollection<Event>();
```
Автоматически обновляет UI при добавлении/удалении элементов.

### INotifyPropertyChanged
```csharp
public class Event : INotifyPropertyChanged { ... }
```
Уведомляет привязанный UI об изменении свойства (например, ParticipantCount).

### Привязка данных (Data Binding)
```xml
<DataGridTextColumn Binding="{Binding Name}" />
```
Связывает колонку DataGrid со свойством модели.

### ListBox с шаблоном
```xml
<ListBox.ItemTemplate>
    <DataTemplate>
        <TextBlock Text="{Binding FullName}" />
    </DataTemplate>
</ListBox.ItemTemplate>
```
